Exercise eXtra1: Dealing Cards

Write a program that deals a pack of cards to four people, so that each person has 13 cards. The cards should be dealt randomly.

You should format the output to neatly display the cards each hand has. An example of a suitable output is: 

north:  KD  QC  7D  7S  8C  3C 10D  KS  JS  AH  JD  8S  5S
 east:  2H  5H  7C  3D  2C  8D  6D  6S  4H  AS  8H  QH  3S
south: 10H  9C  4S  JC  4C  AC  2D  QS  AD 10S  KH  9H  KC
 west:  6H  7H  2S  3H  5D  QD  6C  JH  9S  5C  4D 10C  9D
where the players are called north, east, south and west, as in bridge.

Calculate and display the number of high-card points in each hand. An ace is worth 4 points, King 3 points, Queen 2 points and Jack 1 point. An average hand has 10 points.

Good luck!
