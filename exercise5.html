Exercise 5: Matplotlib

Make sure you have read unit 5 before attempting the following exercise.

The aim of this exercise is to find the area of a circle. We will do this by using uniformly distributed random numbers.
  1. Create a plot showing a square with corners at (-1,-1) and (-1,1), (1,1) and (1,-1).
  2. Add to this plot a circle of radius 1.
  3. Create an array of uniformly distributed random numbers that fall within the square, and add them to your plot.
  4. Calculate the fraction of the points that lie within the circle, and use this information to find the area of the circle. How does your answer compare to the accurate value?
  5. Calculate the percentage error in your value for the circle area, and plot how this error changes as the number of random points is increased.

When you have had a try at this exercise you should you progress to unit 6.
