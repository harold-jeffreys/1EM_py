The following code illustrates a sorting algorithm used by card players. Pick out the second card and then put it in order with respect to the first, then pick out the third card and insert it into the sequence, and so on until the last card. This algorithm uses a list of values and the two types of loops discussed in unit 2. See if you can be your own python interpreter and run this code. 
  
#first generate a list of 20 random integers
#We will come back to this later in the course  
import numpy as np
num_points = 20
data_list = np.random.randint(30,size=num_points)

print("Starting list of numbers")
print(data_list)

for j in range(1, num_points):
    a = data_list[j]
    i = j
    while (i > 0 and data_list[i-1] > a):
        data_list[i] = data_list[i-1]
        i = i -1
    data_list[i] = a

print("Final list of numbers")
print(data_list)
