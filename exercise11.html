Mandelbrot sets

This exercise is based on chapter 5 of Python for Scientists. You have met complex numbers in your Essential mathematics lectures. A complex number $z = x + i y$ can be represented by a point $(x,y)$ on the plane. To determine if a point belongs to the Mandelbrot set $\mathcal{M}$ the result of the iterated sequence $$z_{n+1} = z_n^2 + c$$ is examined. We choose $z_0 = 0$, and set the constant $c$ to the point in the plane we are examining. If $|z_n|$ remains bounded as $n\rightarrow \infty$ then $c$ lies in $\mathcal{M}$. One practical result is that if $|z_n|>2$ for some $n$ then $c$ lies outside $\mathcal{M}$.
  1. Write a function that takes an input of a constant value $c$ and carries out the interation described for up to max_iter = 256 times. It should return the iteration number when $|z_n|>2$. Check your function by showing that $1$ is not in $\mathcal{M}$ but $i$ (i.e. 1j) is. We will call this the escape parameter.

Prime Numbers

The goal of this exercise, is to write a program to implement the Sieve of Eratosthenes, a method to determine the prime numbers. I will describe briefly how the method works, though referring to the Wikipedia entry is also helpful.
The algorithm works by starting from a list of the first n integers. This will then lead to all primes ≤n. The number 1 is immediately discarded from the list, as being a special case. The next number is 2. Since it is still in the list, it must be prime. All multiples of 2 are then crossed off the list. Since they cannot be prime.
The next number, after 2, that is still on the list is 3. That must be prime, but all multiples of 3 still in the list are crossed off. The next number still on the list is 5 (because 4 has already been crossed off). 5 must therefore be prime, and all multiples of 5 are now crossed off, if they have not already been. This continues until you have worked through the whole list (actually you can stop after n/2 and still get the right answer).
Write a program which implements this algorithm.
