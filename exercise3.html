Exercise 3: Functions and Modules

Make sure you have read unit 3 before attempting the following exercise.

  1. Write a python function to convert a temperature in Fahrenheit to its equivalent value in Celsius. The mathematical relationship between the temperature in Celsius and in Fahrenheit is $C=(5/9)(F-32)$ where $C$ is the Celsius temperature, and $F$ is the Fahrenheit temperature.
  2. Write another function to convert a temperature in Celsius to Fahrenheit.
  3. Write two more functions to convert temperature in Celsius to Kelvin, and Kelvin to Celsius.
Create a module containing the functions you have written. Using your module write a program that prints out a temperature conversion table from Celsius to Fahrenheit and Kelvin from $0^\circ$C to $100^\circ$C.

When you are happy with the output from the program, it's time to move on to unit 4.
