Exercise 1: Printing to the screen

In Unit 1 you were shown how to print a message to the screen,and perform some basic calculations.

  1. Write a program to perform some basic operations and print them to the screen. Find the results for
    • An integer divided by an integer
    • A floating point divided by a floating point.
    • Try doing some arithmetic with python at the interactive command prompt by typing python3 into a terminal.
    [Note that python3 returns a floating point for two numbers that are not divisible. This is not the case for many programming languages.]
  2. Calculate the result of the following operations in Python: 1+2-3*4/5
    • Look up the precedence of the operations
    • Modify the result using parentheses () to prioritise calculations
    • Is the result accurate? What has happened to the precision?
