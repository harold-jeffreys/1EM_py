2 - Recursivity

A function is said to be recursive if it calls itself in the execution. A recursive function is usually organised as follows :

  1. Termination criterion : Checking that the function stops at some point
  2. Some computations
  3. Recursion : The function calls itself
  4. More computations
  5. Return a value

Here is a very simple example of recursive function computing the sum from $1$ to $n$ :

def sum_n(n):
    # Termination criterion
    if n==0:
        return 0

    # Computations and recursive call
    return n + sum_n(n-1)

print(sum_n(10))

As you can see here, the function calls itself until $n=0$.

The goal of this exercise is to convert the factorial function of Unit 5 to a recursive version. Here is the code of the function as a reminder :

def factorial(n):
    res = 1
    while n > 1:
        res = res * n
        n = n - 1
    return res

This code corresponds to the definition $n! = 1 \times 2 \times 3 \times ... \times (n-1) \times n$. But we can also define a factorial recursively : $n! = n \times (n-1)!$ and $0! = 1! = 1$. Using this formulation, code a recursive version of the factorial function. Be very careful with the termination criterion or your program will crash after calling itself too many times !

3 - More functions

A ball is placed at $y$ meters from the ground. The ball is dropped with no initial velocity $v_y=0.0\; \rm{ m s}^{-1}$. Using the fact that the gravitational acceleration is $g = -9.81 \;\rm{ms}^{-2}$, we will write an integrator that will compute the position of the ball over time as follows. We are going to use a very simple integrator : Euler's integration method

The Euler method will simply compute the velocity and position at time $t+\Delta t$ using the information at time $t$ :

$y(t+\Delta t) = y(t) + \Delta t \times v_y(t)$
$v_y(t+\Delta t) = v_y(t) + \Delta t \times a$
