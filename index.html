1EM - Essential Mathematics / Computational Laboratory

This is the home page for the computational part of the Year 1 (FHEQ Level 4) Essential Mathematics course.
This course takes place during your weekly scheduled 1EM slot in 32BB03 (Penguin Laboratory). Please come to your scheduled slot, and not one of the other times. There are 11 units and exercises to work through, as listed on the left. The course runs over 11 weeks, so you should be averaging 1 unit per week. This course runs over semester 1, and is examined at the end of semester by a written exam.
If you have questions about the course, you can email me at j.adams@surrey.ac.uk, or find me in Monday 9am computing class. The computing class is delivered with two other members of staff Dr Max Delorme and Dr Francesco Volponi.
This web site does not cover the entirety of the Python language, and is only intended as an introduction. There are a huge number of python programming resources available online, and in the library:
  • J. M. Kinder and P. Nelson, "A student's Guide to Python for physical modeling", Princeton University Press. This is a concise book with several short examples.
  • The official python tutorial
  • Dive into python A more advanced reference.
  • Think Python
  • A byte of python
  • Rhino Python
  • Python-course
Please do make the most of the online resources and those available in the library to help you with your learning. We will use the UNIX operating system to run our programs. The first work unit will start by introducing this here.
