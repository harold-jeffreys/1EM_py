Unit 11: Summary

We finish the course with some more general examples that will help you re-enforce some of the aspects of programming you have learned. First we start with an exercise follows chapter 4 of Kinder and Nelson "A student's guide to Python for Physical Modeling".
A patient with HIV has had an antiretroviral drug administered. We'd like to model the viral load -- that is the number of virions in the blood of the patient. One model for the viral load $V(t)$ a time $t$ after treatment is $$V(t) = A \textrm{exp}(-\alpha t) + B \textrm{exp}(-\beta t)$$ The four parameters $A$, $\alpha$, $B$ and $\beta$ are constants that control the model.
  1. Start by creating a function to evaluate this model of the viral load. Then create a few plots of this function for different values of the model parameters.
  2. Now have a look at some experimental data. The details of the experimental data are here, and the data itself is here. Import this data into an array hiv_data. Separate this data into two arrays and plot it (we did this in unit 6 using numpy's genfromtxt function).
  3. Superimpose the experimental data points on a plot of the model. Adjust the four parameters until you can fit the data to the model. This is quite difficult to do with four parameters! Perhaps try the following: assume that $\beta>\alpha$. How does the trial solution behave at long times? If the data fits this then we can find two of the four constants, then hold them fixed whil adjusting the other two? We can get a better idea about the remaining two by looking at the $t=0$ behaviour.
  4. The latency period of HIV is about ten years. Based on your results how does the inverse of the T-cell infection rate $1/\alpha$ compare to the latency period?
