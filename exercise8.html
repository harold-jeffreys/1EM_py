Exercise 8: I/O

Make sure you have read unit 8 before attempting the following exercise.

    • Write a python program to create a file with the numbers 1 to 10 inside.
    • Read this file and print the contents back to the user.
    • Now ask the user for a number to append to the end of the file, and append the number to the file.
    • Close the file.
    • Now create two functions write_sequence and read_sequence to carry out the operations described in the previous parts of the question.
  2. Open an interactive python session and create a file. In another terminal window try and view the file contents (before closing the file). What do you notice? Try the same but after closing the file.
  3. Download the public data set on UK maximum temperatures from the Met office here. Read this data into python and produce a graph of the maximum summer temperature over time using matplotlib.
When you have done this exercise, it's time for unit 9.
