Exercises 4: Loops

Make sure you have read unit 4 before attempting the following exercise.

1 - while loops

  1. Write a program to:

    • Ask the user to input a integer and store it in a variable called maximum.
    • Initialize another variable total to 0.
    • Using a while loop, ask the user to input an integer.
    • Add this integer to the value of total.
    • Continue asking the user for values as long as total is less or equal to maximum

  2. Copy the following code :

    import random
value = random.randint(1, 100)

    This code generates a random number between 1 and 100 (included) and stores it in the variable value. Using a while loop, ask the user to enter a value and indicate them if their value is bigger or smaller than value. Loop until the user has found the correct value.

2 - for loops

  1. Using a for loop and the range instruction, generate all multiples of 5 between two values entered by the user

  2. Copy the following code :

    first_name = ['luke', 'obiwan', 'leia', 'han']
surname = ['skywalker', 'kenobi', 'organa', 'solo']

    Using a for loop and the zip instruction, generate a dictionary holding the first name of each character as a key and their surnames as values.

  3. Using the lists of the previous question, ask the user for a surname. Using a for loop and the enumerate instruction, print the index of the name in the list. If the name is not in the surname list, then print "not found". For example, if the user enters kenobi the code should return 1 (because surname[1] == "kenobi")

3 - Comprehensions

  1. Code exercise 2 a) using a list comprehension

  2. Code exercise 2 b) using a dictionary comprehension

  3. Copy the following code :

    from math import sqrt
list_1 = [9, 25, -7, 12, 4, -4, 8, 16, -3]

    The function sqrt(x) returns the square root of variable x. Using this function and a list comprehension, create a list holding the square roots of all positive elements of list_1
