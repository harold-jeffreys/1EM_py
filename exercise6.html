  2. Here's a technique to work out the number π to many decimal places. You can show that π can be expressed exactly as $$ \pi=16\tan^{-1}\frac{1}{5} - 4\tan^{-1}\frac{1}{239}. $$ The inverse tangent functions can be evaluated as their Taylor series as $$ \tan^{-1}(x) = x - \frac{1}{3}x^3 + \frac{1}{5}x^5 - \frac{1}{7}x^7 + \cdots $$
    This means that the inverse tangent of 1/5 can be written as $$ \tan^{-1}\frac{1}{5} \approx \left(\frac{1}{5}\right) - \frac{1}{3}\left(\frac{1}{5}\right)^3+\frac{1}{5}\left(\frac{1}{5}\right)^5 - \frac{1}{7}\left(\frac{1}{5}\right)^7 + \cdots $$ and similarly for the inverse tangent of 1/239.

    Write a program using python to calculate an approximate value of $\pi$ using this method, and compare it with the in-built value of np.pi.
