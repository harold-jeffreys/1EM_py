Unit 2: More on variables, flow control operations and loops.

One of the most common tasks of a computer program is to perform calculations. To do this, the programmer must define variables. In mathematics, variables are usually single characters, such as x or t or i etc. When programming, however, you have the flexibility to give your variables names which reflect their meaning. So, you could use x_coordinate or time or radius or counter. You should avoid using reserved words (words that are python commands such as "if", "for" etc.) and you cannot leave spaces (which is why "x_coordinate" has an underscore; "x coordinate" could not be used).

Python has a five standard data types for variables:

  • Numbers
  • Strings
  • Lists
  • Tuple
  • Dictionary
We will focus on the first three types in this course. As we saw in the first unit numbers can be of four different types: signed integers (32-bits of precision), long integers (unlimited precision), floating point numbers and complex numbers (if you haven't already met complex numbers you will meet them shortly in your lectures). We can create these different objects as follows
x_coordinate = 1.1e2
cycles = 102

N_atoms = 120L
complex_impedance = 1+1.2j
We can also convert one type of variable to another using the conversion functions int(), long(), float(), complex(). For example try printing out the value of int(12.3).
We will use string variables principally to create file names, and labels for plot axes. You can create strings using either single quotes, or triple quotes if you need to continue over several lines.
file1 = "data.dat"

message1= """a very long message continuing
over two lines"""
We can combine strings by using the + command, for example
file1 = "file"
file2="0001.dat"

filename = file1+file2
print(filename)
The last data type for a variable we will see is a list. Lists contain a series of values -- for example a list of numbers, or strings.
#a list of polymers
polymer_types=["polyester", "polyvinyl", "polychloroprene"]

#print out the first type
print(polymer_types[1])

#a list of numbers
polymer_length = [100, 1000, 10000]
print(polymer_length[0])
print(polymer_length[2])
Here two lists are defined (polymer_types and polymer_length), and their individual components accessed using the square brackets [ ] appended to the name. The indexing of the components in the list starts from zero, that is the first item in the list is given an index 0, and the second an index 1, and so on.
So far, our example programs have executed each line in turn through the program until the end of the program is reached. Sometimes it is useful to be able to go back and use the same parts of code over and over (e.g. with a new value of a variable) or conditionally execute code based on some test (e.g. testing if a number a user has typed in is in a sensible range). There are a few different ways of doing this, which we will now discuss.

Loops

while loops

The simplest type of loop is a while loop. This is executed while a particular condition is true. For example you might like to loop over all lines in a file while there is still data, or continue a simulation while an end condition has not been met. 
  
time  = 0
while time < 10:
     print(time)
     time = time + 1
print("Exited while loop.")
Note that the block of code that is executed inside the loop is all indented by the same amount. The next part of the code to be executed after the loop has finished is not indented.

for loops

A more frequently used type of loop is the for loop. In this case we have a list of objects to loop over. A simple way to create a list is by using the range() function. If this function is given a single integer as an argument then it generates a list of integers starting from 0 upto but not including this value. This function creates a list For example: 

for x in range(10):
    print("Loop number ", x)

letters = ("a", "b", "c","d")
for x in letters:
    print(x)

if statements

The if statement is used for conditional execution of lines of code. There are various forms of this construct, but the simplest is as shown in the following example. 
  
x=1
if x == 1:
    print("x == 1")
The block of code following the if statement is only executed if the condition is true. If it is not true, then no action is taken. There are many relational operators we can use for comparison

Command

Meaning
a < b true if a is less than b
a <= b true if a is less than or equal to b
a > b true if a is greater than b
a >= b true if a is greater than or equal to b
a == b true if a is equal to b
a != b true if a is not equal to b

We can make the if statement even more powerful as follows 
  


if x == 1:
    print("x == 1")
elif x == 2:
    print("x == 2")
elif x == 3:
    print("x == 3")
else:
    print("x != 1,2,3")
In this code block the print statement is executed only if the corresponding statement is true. If none of them are true then the final statement after the else keyword is executed.
We can terminate a loop early by using the break statement. For example we might look for a particular number in a list: 
  
for x in range(100):
    if x == 72:
        print("Found 72!")
        break
    print(x)
