Preliminary information : Reading input

Programming is a way of automatically repeting tasks and adapting these tasks to different data. Often, it is necessary to ask the user who is running the program to give information. We can do this easily using te input() function. input asks the user to enter some value, and returns it so we can store it in a variable : 

a = input('Please enter a value :')

# Note that the type returned by input is ALWAYS a string :
print(type(a)) # 

# So we need to convert it to an int or a float if necessary
age = input('Hi, what is your age ?')
age = int(age)
print('In ten years, you will be ', age + 10)
