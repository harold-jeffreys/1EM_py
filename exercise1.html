In Unit 1 you were shown how to print a message to the screen,and perform some basic calculations.

  1. You can run python commands in a two different ways. A quick way to investigate the effect of commands is to use the invoke the interactive python interpreter.
    • Open a terminal window, and start the python interpreter by typing ipython3.
    • Enter the print command print("hello world!")
    • Define two integers a = 3, and b = 2.
      [Note that python3 returns a floating point for two numbers that are not divisible. This is not the case for many programming languages. If you require integer division then you must use //.]
    • Calculate results the division and multiplication of these integers.
    • Now try some basic arithmetic with floating point numbers c = 1.23 and d = 3.0e1.
    • Calculate the result of the following operations in Python: 1 + 2 - 3 * 4 / 5.
    • Look up the precedence of the operations -- in what order are the operations calculated?
    • Is the result accurate? What has happened to the precision?
    • Check the type of your variables using the type() function.
  2. Now we will use python by running it on a file containing a program. Create a file fibonacci.py that contains the following python code. 
        
#Fibonacci series  
print("Calculating the Fibonacci series")
	
a = 0
b = 1
	
while b < 20:
	  #print out the current fibonacci series number
	  print(b)
	  #calculate the next term in the series
	  b = a + b
	  #store the previous term in a
	  a = b - a
    Run this program by typing run fibonacci.py in the ipython3 interpreter. Try to understand what it does. We will return to loops later in the course.
