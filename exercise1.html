  3. Create a file fibonacci.py that contains the following python code. 
          
	
	#Fibonacci series  
	print("Calculating the Fibonacci series")
	
	a = 0
	b = 1
	
	while b < 20:
		  #print out the current fibonacci series number
		  print(b)
		  #calculate the next term in the series
		  b = a+b
		  #store the previous term in a
		  a = b-a
    Run this code and try and understand what it does. We will come back to the while loop in the code later in the course.
